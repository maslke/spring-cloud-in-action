## 附录A

附录A中介绍了整本书的demo所用的代码框架，介绍了如何来打包成docker镜像，并在此基础和的运行。

以此为基础，在docker官方网站doc的支持下，学习了docker的相关指令（如何从registry上拉取镜像，如何启动容器，如何关闭容器，如何自己打包镜像并上传到reposity中）。

docker-compose的相关使用，还需继续学习。
